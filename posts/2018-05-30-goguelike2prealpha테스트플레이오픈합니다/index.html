By SeukWon Kang

May 30, 2018

( 동일한 글이 제 블로그 http://kasw.blogspot.com/2018/05/goguelike2-pre-alpha.html 에도 올라가 있습니다. )

Goguelike2 pre-alpha 테스트 플레이 설명

그동안 혼자서 만드는 MMO 라는 목적으로 작업한 goguelike2 가 프리 알파 정도 상태긴 하지만 일단 테스트 플레이가 가능한 수준이 되어서? 공개합니다.

MMO 지만 MMORPG 는 아닙니다. roguelike를 MMO 로 만들어보는 것이라 MMORLG 라고 할수 있습니다.

접속 방법은 브라우저를 사용해서 아래있는 링크로 접속하시면 됩니다. 접속시 10Mbyte 좀 넘는 데이터를 다운 받게 되니 주의해주세요.

다만 firefox 는 느려서 잘 안되는 것 같고 개발에 사용중인 chrome에서는 잘됩니다.

링크에 들어가 보면 3개의 입력창이 있는데 각각을 설명하면

Authkey: 인증키가 있는 경우 입력하면 추가기능을 수행할수 있게 됩니다. (주로 test 기능 ) 인증기는 입력창에 입력 하거나 url에 /?authkey= 뒤에 입력해도 됩니다. (보통 링크에 포함되어 있습니다. )

Tower Number: 1 접속할 타워 번호를 입력합니다. 1번 타워는 default 타워로 표준적인 goguelike플레이를 위한 타워입니다. 2번 타워는 fancy 타워로 goguelike의 기능을 활용한 플레이를 위한 타워입니다.

Player name: Player 접속해서 플레이 할때 다른 플레이어에게 보일 닉네임 을 입력합니다.

타워 번호 ( 1 또는 2 ) 를 고르고 player name 을 입력해서 실행하면 됩니다. 그것도 귀찮으면 그냥 start 를 눌러서 기본 타워 1 에 기본 이름 Player로 시작도 가능 합니다. http://goguelike.iptime.org (링크에 iptime이 보이는 것은 공유기의 ddns기능을 사용하고 있기 때문입니다. ^^) 지속적으로 개발 중이라 새 서버가 배포되면 접속이 끊기게 됩니다.
