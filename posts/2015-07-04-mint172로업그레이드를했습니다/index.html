mint 17.2 로 업그레이드를 했습니다.

By SeukWon Kang

July 4, 2015

2015-07-06 추가 수정 지금 발견했습니다만.. 밝기 설정을 power applet에서 할수 있습니다. 설정 잘되고 문제 없습니다. ;;;

이하 원문입니다.

아침에 확인해보니 업그레이드가 가능하여 17.1 을 17.2로 업그레이드 했습니다.

먼저 데스크탑을 진행하고 문제 없음을 확인한 다음 노트북(인민에어)를 업그레이드 했습니다.

업그레이드는 update manager의 edit 메뉴를 통해서 진행하면 되고 완료후에 재시작을하면 됩니다.

설치후 확인해보니 커널도 업그레이드 되어

3.16.0-38 이 됩니다만 전 이미 3.16.0-39를 설치해서 쓰고 있었으니 큰 차이는 없더군요.

업그레이드 자체는 아주 간단하고 빠르게 이루어 지고 10분도 안걸린듯 합니다.

다만 노트북에선 밝기 조절이 안되는 문제가 (다시)발생했는데 밝기 조절 applet이 없어져서 다시 찾아 설치했지만 조절 되지 않는 문제가 있습니다. ( applet은 다시 지워 버렸습니다. )

여전히 fn키 조합으로 밝기조절은 작동하지 않습니다. ( 무슨 문제인지 잠깐 변화했다가 다시 원래대로 돌아가버립니다. )

이전글인 한성인민에어에서 밝기 조절하기 http://kasw.blogspot.kr/2013/12/blog-post.html

를 참고로 수동 조절을 하도록 해서 일단 눈부신 것은 막았지만 그때 그때 조절하는 것은 좀 귀찮게 되었습니다.

참고로 위치가 바뀌어서 저 글 그대로 해서는 안되고 echo $1 > /sys/class/backlight/intel_backlight/brightness 가 들어 있는 스크립트를 만들어서 sudo로 인자를 주어 실행하는 형태를 사용하거나

아니면 /etc/rc.local 파일 중간에 넣어 부팅할때마다 자동으로 설정되도록 할수도 있습니다. ( $1 대신 적당한 수자 - 저는 300을 사용했습니다. )

아무래도 매번 sudo를 해야 하니 그냥 rc.local에 넣는 것이 편할듯 합니다.
