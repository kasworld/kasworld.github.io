오래된 컴퓨터가 전원이 바로 들어 오지 않는 경우.

By SeukWon Kang

April 25, 2015

집에서 사용하는 컴퓨터가 대략 1년 전부터 전원버튼을 눌러도 켜지는데 까지 시간이 걸리는 증상이 있었습니다.

(오해가 없도록 ) 정확히 말하면 전원 버튼을 눌러도 bios의 post가 바로 시작되는 것이 아니고 수십초(또는 수분) 정도의 시간이 지나야 (메인보드에 전원이 들어오면서 ) post가 시작되는 증상입니다.

저만 그런 것이 아니고 제 와이프가 사용하고 있는 (동일 구매 시점) 동일 사양의 기계에서도 같은 증상이 일어나고 있었습니다.

약 4년 전에 산 intel i2500  cpu + gigabyte p67a-ud3r-b3 인데 슬슬 수명이 다되 가는 것이겠거니 하고 포기 하고 있었습니다.

검색을 해봐도 비슷한 증상을 찾기도 힘들고, 게다가  검색하기도 힘듭니다. 다들 윈도 부팅이 느려요 같은 글들만 올라오니까요.

게다가 일단 전원을 눌러놓고 샤워를 한다던지 , 아침을 먹고 온다던지하면 아무튼 켜지기는 하니까요. ;;;

그런데 갑자기 생각이 떠올라서 bios에 있는 전원 설정쪽을 바꿔 봤습니다.

몇년 전엔가 가전을 포함해서 전원을 꺼도 전기를 사용하는 것에 대한 규제에 따른 기능이 들어간 적이 있었지요.

찾아보니 EuP is a European “green” standard for “Energy using Products”. The 2013 standard says that a device, when turned off shall consume no more than .5 Watts of electricity.

( 위키백과에는 없군요. )

이게 문제였던 것 같습니다.

이것이 켜져 있었는데 이부분의 회로가 오래되어 오작동 하고 있었던것 같습니다.

아무튼 이것을 사용하지 않음 ( 즉 전기를 좀 사용도록 ) 바꿨더니 전원이 늦게 들어오는 문제가 해결되었습니다.

( 적어도 현재 까지는 해결된것 같이 보입니다. )
