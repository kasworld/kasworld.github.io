에이서 a500 안드로이드 패드를 구입했습니다.

By SeukWon Kang

September 28, 2011

지난주 토요일에 택배가 도착했으니 이제 5일되었군요.

에이서에서 1000대 한정으로 가격을 내려서 판매한다고 하는 이벤트( 아마도 안팔려서 내린거라고 생각하긴 하지만 ) 에 낚여서? 어부인의 허가를 받고 구입할 수 있었습니다.

구매당시에 설치되어 있는 안드로이드 버전은 3.0 이며 켜자마자 네트웍을 연결해주면 바로 3.1로 업데이트 됩니다. 그리고 어제 3.2로 업데이트 하더군요. ;;; ( 즉 이제부터 새로 사시는 분은 사서 켜면 바로 3.2로 업데이트 될것입니다. )

3.1을 딱 4일 써본 관계로 자세히는 모르지만 3.1과 3.2의 차이가 미묘하게 있습니다.

( 이 글은 그 차이를 까먹기 전에 기록하기 위한 포스트 입니다 ^^ ) ( 즉 스펙이나 사진, 사용기를 적는 글이 아닙니다. ^^ )

일단 3.1에는 당황스럽게도 microsd 카드를 unmount 할수 있는 메뉴가 없습니다. ;;; 그래서 그냥 빼야 합니다. 그래서 오오 , 구글이 뭔가 훌륭한 기능을 넣어서 그냥 빼도 되게 안드로이드를 만들었나보다 라고 … … 생각했을 리가 없지요. ;;; 3.2에는 설정 - 메뉴 한구석에 unmount 메뉴가 있습니다. ;;; 그리고 상태 막대 ( 화면 아래에 항상나오는 ) 에 sd카드 아이콘이 떠있습니다.

3.1에서는 음악을 듣다가 화면이 꺼지면 현재 곡이 다 플레이 된 후 다음곡이 플레이 되지 않습니다. ;;; 화면을 켜면 ( 락을 푸는 화면으로만 들어가도 ) 다음곡이 플레이됩니다. ;;; 3.2에선 잘 플레이 됩니다. ;;;

추가 수정합니다. : 무슨 관계인지 “화면이 꺼질때 wifi 역시 같이 끄기” 옵션이 켜져 있으면 음악플레이가 멈춥니다. 즉 wifi가 꺼지면 음악도 꺼집니다. ( 이게 무슨 관계일까요? ;;; ) wifi가 켜저 있으면(화면이 꺼지는 경우도) 음악 플레이가 됩니다. 그런데 이경우에 직직거리는 잡음이 끼어 드는 군요. 제가 착각한게 3.2로 바뀌면서 저 wifi 설정이 자동으로 항상 켜짐으로 바뀌어서 였습니다. 이 옵션을 누가 바꿨지? 하면서 원래 제 설정( 화면 과 같이 wifi 도 꺼짐)으로 바꾸니 음악이 멈추는군요. ;;;

그리고 드디어 ( 겨우 5일 된 주제에 ^^ ) 구글 뮤직이 기본 음악플레이어에서 됩니다. .만세 !!!

그리고 설정메뉴에서 터치 감도 설정이 들어갔고. 설정 메뉴의 about에 legal info 가 새로 들어간듯 ( 이건 원래 있었는지 기억이 잘 안납니다. )

그외에는 눈에 확 띄는 변화는 없는 듯 합니다.

그냥 3.1을 급히 내느라 생긴 문제들을 해결한 게 3.2인듯합니다.

그런데 3.0,3.1,3.2 가 다 허니컴 이더군요. ;; ( 2.x 대에선 다 이름이 다르지 않았던가요? )
