sublime text 2 에디터를 구입했습니다. - 수정했습니다.

By SeukWon Kang

January 20, 2013

요즘 프로그래밍용 텍스트 에디터의 대세^^인 sublime text 2 를 설치해서 써보고 있습니다.

여러 플랫폼(windows, linux, max os )을 지원 하고 라이센스도 머신 단위가 아니고 구매자 단위로 주어지는 등 꽤 맘에 드는 부분이 많은 에디터 입니다.

보면 볼수록

도스 시절의 강자였던 brief ( http://en.wikipedia.org/wiki/Brief_(text_editor) )

win16 시절의 강자였던 codewright ( http://en.wikipedia.org/wiki/CodeWright )

들의 정통 후계자란 느낌이 들어서 구매하는 것을 심각하게 고민하기 시작했습니다.

심지어 가격도 전성기 시절의 위 에디터 가격 보다 싸군요. ( biref : $199, codewright $250 , subslime text $59! ) ( biref는 개인구매, codewright는 회사것을 사용 했었지요. )

각 에디터가 모두 맛이 간 이후로 ( 특이하게도 둘다 borland 에 인수 되었습니다. 지금은 다시 분리된듯 하지만.. ) 다시는 closed source editor는 안쓰겠다고 vim에 정착 했었지만..

아래의 비디오 강좌를 들으면서 옆에 실행시켜놓고 실습해가면서 마음을 굳히고 결국 페이팔 가입까지 해가면서 질렀습니다.

정말 오랫만에 보는 세세한 부분까지 사용자의 편의성을 신경쓴 제품이라고 생각합니다.

수정사항.. subslime 이 아니고 sublime 이더군요. 아~~ 창피..

sub-slime 이라고 생각해서 개그스런 이름이네 라고생각 했었는데 말이죠.. 그게 저 위에 있는 brief(속옷)editor 를 만든 회사는 Basic Reconfigurable Interactive EditingFacility 의 약자라고 주장했지만  회사이름이 UnderWare Inc, 였단 말이지요. ;;;  그래서 에디터 만드는 넘들은 다들 개그를 좋아해.. 하고 생각 하고 있었는데;;; 

sublime(절묘한,숭고한) 이라니.. 재미 없는 넘들.. ( 애초에 모르는 단어였지만;;;;)

아래 링크들은 구글링하면 30초면 찾을수 있는 것 이지만 기록과 정보용으로 적습니다.

에디터 홈페이지 http://www.sublimetext.com/

문서는 http://www.sublimetext.com/docs/2/

비디오 강좌 - 영어도 별로 어렵지 않고 좋은 내용이 많으니 꼭 들어보시기를 강추합니다. https://tutsplus.com/lesson/installation-and-base-settings/

위 비디오에도 나오지만 사용하기위해선 꼭 필요한 플러그인 관리자 http://wbond.net/sublime_packages/package_control/usage
