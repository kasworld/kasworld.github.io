goguelike 개발진행 - 중2병스러운 장비이름등

By SeukWon Kang

August 3, 2018

약 2주간  진행된 작업 정리 입니다.

아래 동영상은 최신 버전goguelike 의 4배속 플레이 입니다.

오류가 발생했습니다.

www.youtube.com에서 동영상을 시청하거나 브라우저에서 자바스크립트가 사용 중지된 경우 사용 설정하세요.

보이는 변동사항으로는

장비 슬롯의 이름과 기능이 바뀌었습니다. 타일 형태 : 슬롯 이름 : 전투기여 { Weapon : Attack [ Shield : Defence ^ Head : Defence % Armor : Defence

  • Gauntlet : Defence ; Boots : Defence o Ring : Attack
  • Amulet : Attack 입니다.

장비 생성시 부위 별로 능력 수치가 다르게 생성됩니다. “Weapon”, 2.0 “Shield”, 1.0 “Head”, 0.5 “Armor”, 1.0 “Gauntlet”, 0.5 “Boots”, 0.5 “Ring”, 0.5 “Amulet”, 0.5 의 비율로 장비 수치가 생성됩니다.

장비의 이름이 생겼습니다. 이름이 좀 긴데 “최초소유자이름 +  형용사 + 재질 + 이름” 의 형태입니다. 보시면 중2병 기운이 풀풀 나는 것을 알수 있을겁니다. ;;

폰트 크기를 화면 크기와 연동하도록 하였습니다. 모바일과 데스크탑에서 적절한 가시성을 가질수 있기를 희망하고 있습니다.

메시지와 인벤토리목록에 스크롤바를 넣었습니다. 기존에는 내용이 많아지면 하염없이 길어지는 문제가 있었습니다.

도움말과 표시되는 내용들을 좀더 알아볼만하게 수정하였습니다.

하나의 플로어를 완료(모든 타일을 확인) 하고 나면 그 플로어 내에서 자유롭게 teleport 할수 있고 또 다른 플로어에서 완료한 플로어로 아무때나 이동할수 있게 되었습니다. 완료한 플로어가 생기면 화면에 기능 버튼이 생깁니다.

포탈과 샵등의 상호작용 개체위에 올라가면 자동으로 기능 버튼이 생기거나 ( sell 버튼 ) 활성화 ( enter portal 버튼 ) 됩니다. 그리고 개체의 메시지가 표시됩니다.

화면에 보이는 글자의 색을  faction color 에서 그냥 흰색으로 바꿨습니다. ( 가독성 향상 )

현재 플로어에서 발견한 IaO 목록이 표시됩니다.

포션, 버프, 서버 AI 등의내부 개선이 있었습니다.
