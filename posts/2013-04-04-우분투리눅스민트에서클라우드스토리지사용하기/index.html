우분투/리눅스 민트에서 클라우드 스토리지 사용하기.

By SeukWon Kang

April 4, 2013

전에 쓴 이곳 저곳에서 개인데이터를 안전하게 접근 사용하기. 글에서 NAS가 어쩌고 하긴 했지만.. 

현실은 언제나 가난한 월급쟁이인지라. ;;; 

리눅스에서 사용가능한 클라우드 싱크 스토리지 들 몇개를 설치 해봤습니다. 

현재까지 테스트해본것들은 

구글 드라이브

네이버 ndrive 

다음 클라우드

dropbox

box.com

정도인데.. 

이중에서 리눅스에서 잘되는 것들은 

다음, dropbox, box.com 이군요. 

dropbox 야 리눅스를 정식으로 아주 잘 지원하니 홈페이지에서 다운받아 설치/로그인 하면 아무 문제 없이 사용할수 있습니다. 

용량만 좀 많이 주면 고민할 필요가 없을텐데 말이죠.. 

특이하게도 구글 드라이브는 리눅스에서 편하게 쓸수 있는 방법이 없습니다. 

( 혹시 있으면 알려주시면 감사. )

네이버야 애초에 기대하지도 않았고.. 

특이하게도 다음은 나름 정식으로 지원하는데.. ( 홈페이지에 리눅스용 설치가 있습니다. )

아니나 다를까 x86용 32bit 용만 있습니다. 

하지만 검색해보면 이 패키지를 64bit용으로 변환 해서 설치할수 있게 만든것 또는 만드는 법 이 있으므로 그대로 따라 하면 잘 됩니다. 

저는 여러 곳 중에서 

http://logon.tistory.com/107

을 참고하여 설치하였습니다. 

리눅스 민트는 뒤의 dconf-tools 부분이 필요 없습니다. ( 망할 unity ^^ )

그리고 box 설치는 

http://www.liberiangeek.net/2012/03/automatically-mount-box-cloud-storage-in-ubuntu-12-04-11-10/

를 따라 하면 잘되는데..

자세히 보면 

box 가 webdav접근 을 열어 주고 있으며 리눅스에서 webdav를 마운트 해 쓰는 법이란걸 알수 있습니다. 

그이야기는 집 서버에 webdav를 설치하고 포트/인증설정을 하면 어디서나 박스와 같은 방법으로 접근 가능하단 이야기기도 합니다..

뭐. ftp를 열거나 sshfs 를 사용하는것과 별 차이는 없겠지만요. ^^

박스를 사용해볼 생각을 한이유는 기본 제공 용량은 5G로 그저 그런 용량이지만 현재 프로모션으로 LG폰이나 소니 엑스페이라 폰 사용자가 박스 앱을 다운 설치하면 50G를 준다고 해서 사용하기로 했습니다.  ^^

( 전 둘다 있군요. ;;; 둘다 설치하면 100G를 줄려나요? ;; )

다만 box의 문제는 파일 한개의 최대 용량 제한이 있습니다.
