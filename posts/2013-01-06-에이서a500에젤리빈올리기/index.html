에이서 a500에 젤리빈 올리기.

By SeukWon Kang

January 6, 2013

뱀띠 새해를 맞이하여 오랫만에 a500에 커펌질을 했습니다.

기존에 쓰던 ics 4.0.3 lightspeed 커펌이 슬슬 지져워 지기도 했고 해서

우리의 호프 xda 포럼을 뒤적 뒤적 해서 http://forum.xda-developers.com/forumdisplay.php?f=1136

적당한 커펌을 선정 했습니다. http://forum.xda-developers.com/showthread.php?t=1935582

엄청나게 긴 이름을 가진 CM10-FLEX_V6Supercharged_Rev2.3_GappsIncl_Public  을 사용 하기로 합니다.  젤리빈 4.1.2 버전입니다. 아무 생각없이 ( 워낙 오랫만이라 뭐를 어떻게 했었는지 기억도 안나고 해서 ) 위 커펌을 받아서 sd 카드에 넣고 리커버리 모드를 들어가서 install zip file을 해봅니다.

업데이트가 시작하는 척 하더니 먹통이 됩니다.

땀이 삐질…

바짝 긴장해서 글을 열심히 정독을 시작합니다. ;;;;;

붉은 글씨를 읽어보니 For ICS bootloader only. 란 문장하고 

Aroma installer is tested and confirmed working on the following recovery’s: ezterry’s A500PubRecovery or  TWRP 2.1.3 (not 2.2…)  **or ** Thor 1.7.3  or  ra-recovery 3.17 (recovery-ra-iconia-3.17-gnm)  란 문장이 보입니다.

하도 예전이라 기억이 나진 않지만 아마도 둘다 해당사항이 없는 듯 합니다.

즉 둘다 해야 한다는 것.. (한숨..)

일단 좀더 쉬워 보이는 복구 모드 설치부터 합니다.

기억을 되살려서 기초 강좌부터 차근 차근 읽어봅니다. http://forum.xda-developers.com/showthread.php?t=1205204

복구 모드 설치는 구글 마켓에서 Acer recovery installer 를 받아서 설치하면 되는 군요. 설치하고 실행해서   **Thor 1.7.3 **를 선택했습니다.

실은 이전에 여기 까지 오는데 다른 삽질을 더 했지만 생략. ( 복구 영역을 깨 먹었다던지 .. ;;; )

그리고 마의 부트 로더 설치를 합니다. http://forum.xda-developers.com/showthread.php?t=1622425

a500 설정 메뉴에서 usb debugging을 켜고 일단 a500과 pc를 연결해서 usb드라이버를 설치 합니다.

에이서 사이트에가서 pc 용 설치 프로그램 (USB_Acer_1.00.0524_A30HA31HA32H_A.zip) 을 받아 설치 합니다. http://us.acer.com/ac/en/US/content/drivers

그다음엔** **SBK  (16진)문자열을 찾아야 합니다. ( secure boot key ? ) 이 문자열은 cpuid를 사용해서 얻을수 있는데  cpuid를  sbk로 변환하는 것은  http://a500bootloaderflash.tk/sbkcalc/ 를 통해서 가능합니다.  **cpuid는 각 기계 마다 모두 다르므로 각자 얻어야 합니다. ** 이 cpuid는  adb devices 명령으로 얻을수 있는데 이 adb.exe는 위에 설치한 에이서 프로그램 에 들어 있습니다.

제경우는 33c11c743bf9257 recovery 과 같은 결과를 얻었고 33c11c743bf9257 을 위 웹페이를 통해서 sbk로 변환하면 0x8D15D201 0x143E8901 0x4B766D03 0x3E572103 을 얻습니다.

이제 APX-FLASH-Tool & Manager 를 사용해서 부트로더를 설치합니다.  부트로더는 여러개 중에서 선택할수 있는데  저는  V8-Thor_1.7.3_recovery.a500apx 를 사용했습니다.

위 프로그램을 받아서 설치 했으면 이제는 조심조심 부트로더 설치에 들어갑니다. 이 과정에서 실수하면 기계가 벽돌이 되니 아주 조심해서 진행합니다. 

1. Settings -> “Applications -> Development -> USB debugging 을 켭니다. ( 한글로는 알아서 찾으시길.. ) 2. A500APXFlash.exe 를 실행합니다. 3. 다음 다음으로 진행하다 sbk를 입력하는 부분에서 sbk를 입력합니다.  4. “Use an A500 APX Flash Tool Package.(.a500apx)” 를 선택합니다.  5. 파일을 선택하는 창에서 저는 아까 받은 V8-Thor_1.7.3_recovery.a500apx 를 선택했습니다.  6. Automatic Method 를 선택합니다. 

7. “Enter APX Mode” 를 선택합니다. - a500 화면에 apx mode어쩌구 가 나옵니다.  8. 다음으로 진행합니다.  “Done! Press and hold power to switch off then power back on!" 이 나오면 끝난 것입니다.  부트 로더 설치가 끝나면 복구 모드에 들어가서 커펌을 설치하면 됩니다.  주의 할점은 부트 로더와 펌웨어는 둘다 ics 이후 버전이거나 둘다 허니컴 버전으로 일치 해야 합니다.

그런데 저 커펌 설치 무지하게 화려합니다. ( 설치 화면이 ;;; )
