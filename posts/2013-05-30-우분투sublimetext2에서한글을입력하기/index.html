우분투 sublime text2에서 한글을 입력하기.

By SeukWon Kang

May 30, 2013

안되는 거라고 생각하고 포기 하고 있었는데 같이 일하는 동료의 제보로 약간 불편하지만 일단 입력이 가능하게 되어 기록하기 위한 포스트. ;;;

일단 한글 입력이 안되는 원인은 ibus의 버그 때문 이라고 합니다.

실제로 windows/OSX 에서는 한글 입력이 잘된다고하는 군요.

그럼 ibus가 아니고 다른 IME를 쓰면 해결 되지 않을까? 에서 시작합니다.

한줄 요약하면

http://www.interoasis.net/post.php?No=15

에 나와있듯이

sudo apt-get install uim uim-hangul im-switch -c /* 한글입력기 변경 */ 그리고 X 재시작 ( 로그아웃 또는 재부팅 )

단 두줄만 입력하면 간단히 해결 됩니다.

하지만 함정이 있는데.. 위 형태로 api-get을 사용해서 설치하면 아무 문제 없이 잘 설치되며 이 상태로 sublime text2에서 한글 입력도 가능하게 됩니다만..

synaptic 을 사용해서 설치하면 제대로 되지 않습니다. 회사에선 간단히 성공해서 집에와서 설치하다가 잘 안되서 고생했는데 원인이 저거 였습니다.

자세히 보니 synaptic으로 설치하면 apt-get으로 설치할때 설치되는 패키지의 일부가 설치 되지 않는 군요. ;;;

아무튼 이과정을 거치고 나면 한글 입력이 잘됩니다.

단 sublime text 2에서는 한글 입력의 첫 글자가 화면에 보이지 않습니다. 아마도 현재 입력중인 글자가 안보이는 거라고 생각하는데.

이거 예전 윈도우에서 영문 프로그램에 한글 입력기 설치해서 한글 입력할때 많이 보던 증상이군요. ;;;

그래도 입력이 되는 것 만으로도 기뻐하고 있습니다.
