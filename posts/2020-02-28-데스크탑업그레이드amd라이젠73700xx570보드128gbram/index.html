데스크탑 업그레이드 AMD 라이젠 7 3700X + X570 보드+ 128GB ram

By SeukWon Kang

February 28, 2020

요즘엔 노트북도 데스크탑못지 않게 빠르니 앞으로는 데스크탑은 사지 않고 노트북만 쓰겠어! 라고 생각했던 적이 저에게도 있었습니다. ^^;

2년 반 정도 잘 사용해온 한성 노트북이 goguelike 플레이를 녹화 하는데 버벅이더군요.

인텔 i7-7700HQ (4코어 8쓰레드) cpu 에 24GB 램 1T nvme 디스크 로는 모자란가? 하고 호시탐탐 업그레이드를 위한 조사를 몇달간! 진행했습니다.

업그레이드를 하면 적어도 2배는 빨라져야한다. 라는 기준을 세우고 검색을 해봤으나 현재 노트북에서는 극도로 많은 비용( 300만원 이상 ) 을 지불 하기 전에는 2배성능의 기계를 살수 없다는 것을 알았습니다.

하반기에 나온다는 노트북용의 AMD 라이젠 4800H 정도면 원하는 성능을 내줄수 있을것 같아서 느긋하게 기다릴 생각을 하고 있었습니다.

제 처가 사용하는 노트북 을 AS 보내야 할것 같은 정황이 생겨서 이를 대신할 윈도우 기계가 필요하게 되었습니다.

AS기간이 끝나는 5월 이전에 처리 해야 하니 그전에 여유 기계를 하나 준비해야 하는 핑계가 생긴거지요.

그래서 제가 쓰던 노트북을 임시로 쓸수 있게 윈도우로 바꾸고 ( 전 리눅스를 씁니다. ) 저 새 기계를 하나 장만하도록 이야기가 되었지요.

그래서 급히 기존에 놀고 있던 데스크탑의  cpu/ram/board 를 업그레이드 하게 되었습니다.

기존 데스크탑은 11년 1사 분기에 나오자 마자 구입한 Intel® Core™ i5-2500 (4코어 4쓰레드) 에 16GB ddr3 ram 입니다. 따라서 cpu/ram/board를 모두 바꿔야하는 거지요.

기존에 쓰던 케이스, 파워, 그래픽카드(nvidia 750ti) 는 그대로 쓰기로 했습니다.

업그레이드 후 간단히 성능 테스트를 해보니

7z 점수가 ( 7z b ) i5-2500 : 12000 i7-7700HQ : 19000 Ryzen 3700X : 57000

정도로 나와 3배정도 빨라진것같습니다.

kazam ( 리눅스용 화면 녹화 프로그램 ) 으로 4k 녹화를 해봤는데 아주 잘됩니다.

기존에는 FHD 녹화도 버벅이던거에 비하면 아주 만족스럽군요.
