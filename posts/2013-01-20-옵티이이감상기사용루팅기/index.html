옵티이이 감상기(사용, 루팅기)

By SeukWon Kang

January 20, 2013

넥서스4와 옵티머스G 그리고 옵티머스 lte2를 놓고 무엇을 살까 고민하다가 AS 문제와 가격적인 문제로 ( 옵티이이가 그래도 좀 쌌습니다. ) 옵티이이를 선택했습니다.

그리고 이제 2주가 약간 넘어가니 사용기? 내지는 감상기를 적어 봅니다.

장점 교환식 배터리 외장메모리 지원 배터리를 빼지 않고 외장 메모리 교체 가능 - 즉 핫 스왑 가능 ( 뒷 판을 열여야 하긴 하지만 뒷판만 열면 바로 넣고 뺄수 있습니다. ) 최신 OS 지원 ( 4.1.2) 꽤나 넉넉한 램 ( 2G) 적당한? cpu/gpu 사양 ( 듀얼코어 a15 - 1.5Ghz ) - 최신보다 한그레이드 낮습니다.

단점 부트로더가 락되어 있다. - LG폰들은 다 그렇답니다. OTG 미지원 - LG폰들은 다 그렇답니다. 버튼식 home 키 사용감이 별로임 ( 레이 홈키가 정말 좋은 것이더군요. ) 루팅을 해도 지울 수 없는 통신/제조사 s/w 들 최신에 비해 약간? 떨어지는 스펙 - 대신 발열과 배터리에 조금 이득이 있을지도. 마이너한 기종이라 액서세리가 적습니다. - 전 거의 안쓰는 편이라 신경 안씁니다만..

개인취향 각진 디자인 h/w (터치,버튼식) 기능키들 ( back,home,menu ) 존재 쓸데 없는 DMB ;;; - 좋아 하는 분들도 있겠지만 전 차라리 라디오. 쪽입니다. 1280x800이나 768이 아닌 720해상도   가로 폭이 좁아져 그립감은 향상되지만  해상도는 다다익선..

카메라 - 속도냐 화질이냐 중에서 선택하라고 하면 속도를 선택할 사람이 적을 듯.   화질이 떨어지는 것이 제로랙 카메라를 위해서 라는 루머가 있는데 확실히 hdr 모드(트릭이지만) 을 선택해서 사진을 찍어보면 화질이 올라가긴 한다.   하지만 사용자에게 제로랙을 안쓰고 화질을 선택할수 있는 정상적인 UI 를 제공하지 않는것은 잘못. 미니 유심 - 멀쩡한 유심을 칼로 자르느라 고생했습니다.

감상 요즘 안드로이드/ios 에서 h/w, s/w 의 발전을 보면 지난 10년간 intel과 MS가 독점을 하면서 발전이 얼마나 정체되어 있었는지를 알게 되더군요. 아직까지는 안드로이드에서 2gbyte 정도의 메모리면 별로 부족함이 없습니다만 아마도 곧 더 큰 메모리를 요구 하게 될텐데 ( 게임과 웹 브라우징을 생각하면 ) 4G 메모리, 그리고 그다음은 64bit!!! arm cpu(이건 이미 발표되어있지요)와 64bit 안드로이드 OS 의 등장이겠네요. 고해상도 화면과 빠른 CPU ,대용량의 램은 현 노트북,데스크 탑의 위치를 금방 위협할것 처럼 보이기도 합니다. 실제로 안드로이드 태블릿에 블루투스 키보드 , OTG USB 를 연결해 쓰다보면 pc/notebook 이 필요 없구나 하는 생각이 듭니다.   제가 a500을 그렇게 씁니다. 덕분에 제 X61T(고해상도 태블릿모델입니다.)이 서랍속에서 자고 있지요. 애초에 같은 타겟으로 크롬북같은 것도 있구요.

젤리빈 업데이트 받자마자 젤리빈 업데이트를 lg 홈페이지를 통해서 했습니다. 기기내의 설정메뉴의 업데이트나 OTA로는 업데이트 되지 않더군요. 덕분에 업데이트가 꽤 귀찮습니다.

루팅과 커스터마이징 국내 전용 모델인 관계로 XDA를 찾아봐야 필요한 정보가 없습니다. 대신 국내 사이트 를 찾아보면 기본적으로 필요한 정보를 찾을 수는 있는데. 루팅를 하는 것은 http://redyeller.tistory.com/90 을 참고하면 될것 같고. 네이버 카페를 가입해서 http://cafe.naver.com/opte2.cafe 그래도 조금이나마 필요한 정보를 얻을 수 있습니다.

부트로더 언락이 안된 관계로 아직 커스텀 롬들은 활성화가 안되어 있습니다. 국내 전용 모델 문제와 겹처서 아마 앞으로 힘들듯 합니다.
