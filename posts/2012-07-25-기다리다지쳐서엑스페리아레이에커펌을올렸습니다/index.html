기다리다 지쳐서 엑스페리아 레이에 커펌을 올렸습니다.

By SeukWon Kang

July 25, 2012

소니에서 아이스크림 샌드위치를 올려준다고 약속한 날짜가 5월달에서 7월로 밀리더니 아직도 소식이 없어서 포기하고 XDA 를 뒤적뒤적 해서 ICS 커펌을 올려버렸습니다.

이 글은 그 기록입니다.

소니 안드로이드 폰은 모두(?) 부트로더 락이 걸려 있는 관계로 커펌을 올리려면 언락 과정을 거쳐야 합니다. http://optimistsc.tistory.com/22 위 링크 내용을 간단 요약하면 소니 언락 홈페이지 가서 내 폰의 IMEI 를 입력 언락 코드를 받는다. ( 이과정에서 AS 보장 포기등을 동의 해야 합니다 ) 이 코드를 사용해서  fastboot  틀로 언락을 한다.  폰을 fastboot 모드로 연결합니다. 꺼진 상태의 폰을 볼륨업키를 누른 채로 USB로 연결합니다. 연결이 제대로 되면 홈키 테두리에 파란색 불이 바로 들어옵니다. 안드로이드 usb 드라이버를 설치한후  다음을 실행합니다.  fastboot -i 0x0fce oem unlock 0x언락코드

그런데 이 USB 드라이버 설치가 꽤나 까탈스럽습니다. 기본적으론 PC companion 을 설치하면 되는 것 같은데 windows 7 64bit라 그런지  가장 중요한 마지막 드라이버 하나가 자동설치가 안되더군요.  결국 1시간 이상 고생하다가 강제로 설치를 했습니다.

이렇게 언락이 되면  커펌을 설치할수 있게 되는 데, 커널과 OS ROM? 을 각각 선택해야 합니다.  롬은 아래 것을 선택했고 http://forum.xda-developers.com/showthread.php?t=1535971 커널은  http://forum.xda-developers.com/showthread.php?t=1544513 에가서 고르시면 됩니다. 제가 고른 것은 15/07 [ xKernel v8.0.22 ] - ICS Based Rom 입니다.  이 과정에서도 커널과 ROM의 버전을 일치 시키지 않으면  즉 둘다 ICS 거나 진저/프로요 거나  폰이 부팅이 되지 않습니다.  이걸 몰라서 ROM이 그대로인 상태로 커널만 올렸다가 무한 부팅에 들어가서 살떨리는 경험을 했습니다.  아무튼 커널의 설치는  flashtool http://forum.xda-developers.com/showthread.php?t=1351324 사용하고 ROM의 설치는 CWM 을 사용합니다. CWM은 미리 설치 되어 있거나 아니면 CWM이 포함된 커널을 설치하면 되는듯 하며 CWM 모드로의 진입은 부트중 커널 로고가 나올때 볼륨다운 키를 연타하면 됩니다.

아무튼 중간 삽질 + 롬다운로드 시간 해서 3시간 이상걸린것 같습니다.
