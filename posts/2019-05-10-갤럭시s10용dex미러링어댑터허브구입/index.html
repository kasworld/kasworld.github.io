갤럭시 S 10+ 용 Dex/미러링 어댑터? 허브 ? 구입

By SeukWon Kang

May 10, 2019

올해 초에 약 3년간 쓰던 갤럭시 s7 edge를 (업데이트가 종료 되었다는 핑계로 ) 사용종료 하고,  S 10+을 (사전예약으로!!!) 구입했습니다.

s7 은 낀 세대라 모니터 연결이 불가능한데 s8 부터는 dex 라는 것을 지원해서 유선으로 모니터를 연결할수 있다는 것을 알고 있었기에 S10+을 사면서 dex란 것을 사용해 봐야겠다고 생각하고 있었습니다.

간단히 조사해 보니 S10+ 에서는 그냥 USB type C to HDMI 케이블만 연결하면 dex를 사용할수 있다고 ( 이전 모델들도 파이 업데이트 하면 가능한듯 ) 해서 고민이 시작 되었습니다.

본격적인 USB type C 허브 ( hdmi , Lan , PD 충전, usb hub, SD card reader 등이 포함된 ) 를 구입 할것인가 (비쌉니다. ) 아니면 걍 USB type C to HDMI 변환 케이블 을 구입할것인가. ( 쌉니다. ! )

근 두달간 고민하다 적당히 기능이 적지만 그래도 기본 기능은 있는 것 같은 그리고 저렴한 것을 질렀습니다.

http://prod.danawa.com/info/?pcode=6558962&cate=11316923 아이존이십일 USB Type C to HDMI 컨버터 (EZ-MH3) 이란 것인데 hdmi , pd 충전 , usb 3.0포트 1개 ( 즉 허브 기능 없음 ) 만 있는 것입니다.

가장 중요한 장점은 가격으로 그냥 USB type C to HDMI 변환 케이블 과 가격 차이가 별로 없습니다.

방금 도착해서 테스트 해보니 미러링 , dex , 충전 다 잘되는 것 같습니다. usb 포트가 1개 인것은 원래 PC사용할때도 ( 노트북이라) usb 3.0  허브로 키보드 마우스를 연결해 쓰고 있었기때문에 쓰던 허브를 노트북에서 뽑아서 사용하니 문제 없이 작동하는 군요.

일단 작고, 가볍고 싸고 , 가장 기본 적인 기능이 작동해서 만족하고 있습니다. ( 쓸데가 있을지는 모르겠지만 ^^ ) 참고로 해상도는 dex 모드에서는 ( 어디가 문제인지는 모르겠지만 ) 최대 1920x1200 까지만 지원하는 것으로 나옵니다. 연결된 모니터는 4k ( 3840x2160) 을 지원 하는 모니터고 hdmi 1.x 포트에 연결한 상태 였습니다. 모니터에 한개만 있는  hdmi 2.0 포트는 PS4 pro  가 연결되어 있는 상태로  , PC(노트북) 는 DP 포트에 연결되어 있습니다.

그리고 전원을 연결하기전에는 미러링 모드로만 가능하고 전원을 연결하니 dex 모드로 전환 가능한 UI가 나옵니다.
