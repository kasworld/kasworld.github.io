한성 X57K BossMonster Lv.74 구입,사용기

By SeukWon Kang

July 9, 2017

급히 쓸 일이 생겨서 새 노트북 구입했습니다.

지난주 화요일에 도착했습니다.

한성컴퓨터 X57K BossMonster Lv.74 (SSD 120GB)

저번 한성 노트북을 부셔먹긴 했지만 전 기본적으로 한성 노트북이 꽤나 맘에 드는 관계로 또 한성 노트북을 샀습니다.

( 가격도 싸고, 불필요한 windows 비용 안내도 되고, 쓸데없는 ODD 도 없고 해서 저한테는 딱입니다. )

i7 에 8G ram 1050 4G , 120 ssd 를 달고 80만원 대 입니다.

집에서 쓰는 데스크탑보다 사양이 좋네요. ( ram , ssd 가 작지만 업그레이드 가능하니 )

받자마자 mint 18.2를 설치하고 이런 저런 설정을 했는데. 기대이상으로 맘에 들어하고 있습니다.

요즘엔 당연하지만 리눅스에서 대부분의 장치가 잘 인식되고 잘 작동 됩니다.

( 백라이트, 무선랜, 사운드 )

다만 키보드 백라이트는 어떻게 해야 할지 잘 모르겠네요.

신기한 것은 배터리가 분리형! 이군요. 아직도 이렇게 나오는 노트북이 있다고는 생각도 못했습니다.

터치패드도 아주 잘 작동하고 두손가락 스크롤등 설정 작동도 잘됩니다.

이정도면 마우스 없이도 쓸만할것 같습니다.

nvidia optmus 가 들어 있어서 살짝 걱정했는데 큰 문제 없이 화면 잘나오고 외부 모니터 연결 잘 됩니다.

다만 그래픽 카드 스위칭은 여전히 문제인게

nvidia 관리 프로그램에서 intel/nvidia 를 스위칭 할수 있는데

일단 logout/login 을 해야 적용됩니다.

그리고 intel로 바꾸면 OS 종료하다가 ( 정확히는 x 를 종료하다가 인것 같습니다. )

멈춰 버리고

nvidia로 설정하면 아무 문제 없는 것 같이 보이는데

전원을 연결하지 않은 채로 노트북을 켜면 x 가 시작하다가 멈춰 버립니다.

구매 목적이 들고 다닐수 있는 ( 들고 다니며 쓰는게 아니고 ) 리눅스 머신 이었기 때문에 저에겐 큰 문제가 되지는 않는데 다른 용도라면 역시

논옵티머스가 되는 노트북을 사는 것이 좋을것 같습니다.

전 그걸 위해서 더 무거운 무게(400g) 와 더 비싼 가격 ( 두배 정도 ) 을 지불할 생각이 없어서 이 것을 샀습니다.

외부 모니터를 hdmi, mini displayport x 2 연결 할 수 있는데 dp는 당연히 4k 60hz 가 될텐데 htmi 는 2.0 이 아닙니다. 4k 30hz 만 지원 하네요.

전 4k 60hz 가 필요한 관계로 mini dp 케이블을 주문 했습니다.

아무튼 급히 사서 급히 설정하고 필요 목적을 달성 ( 리눅스 에서 만든 게임 프로토 타이핑 시연 - 게임이 리눅스용인게 아니고 시연용 프로토 타이핑이 리눅스용 )했습니다.

그래서 이제 업그레이드를 해야지 하고 하판을 뜯으려 하판 나사들을 다 제거 했는데 하판을 여는 법을 모르겠습니다.

이전 노트북은 힘을 전혀 주지 않아도 하판이 들렸는데 이녀석은 나사를 다 뺐는데도 하판이 들리지 않네요.

함부로 힘을 줬다간 어딘가 부러 질것 같아서 일단 다시 나사를 다 조여 두었습니다.
