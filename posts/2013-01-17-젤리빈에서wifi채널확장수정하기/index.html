젤리빈에서 wifi 채널 확장/수정하기

By SeukWon Kang

January 17, 2013

에이서 아이코니아 탭 a500 에 젤리빈 커펌을 올린 후 http://kasw.blogspot.kr/2013/01/a500.html

오늘 아침에 일어나보니 a500이 무선랜이 끊어 져 있더군요. 어라 공유기는 이상없는데 하고 공유기 관리 페이지를 들어가보니 wifi 사용 채널이 13으로 되어 있는 것을 보고 갑자기 생각이 났습니다.

http://en.wikipedia.org/wiki/List_of_WLAN_channels

wifi 2.4 GHz (802.11b/g/n) 은 각 나라별로 사용하는 채널이 다르며 미국이 가장 좁은 범위를 사용합니다.

그래서 해외쪽 커펌을 올리다 보면 wifi 채널이 좁게 설정된 경우가 있는데 이경우 우리나라 공유기가 그 밖의 범위 채널을 쓰도록 설정되면 wifi가 않잡히게 됩니다. ;;;

이거 때문에 예전에 wibro egg쓸때도 고생한 기억이 나는데요 . 표면적으로 드러나는 증상이 wifi가 되다 안되다 하기 때문에 뭐가 문제인지 알기 어렵습니다.

아무튼 루팅된 안드로이드라면 build.prop을 고치거나 ( 2.x) , sqlite system db( 3.x ,4.0.x ) 를 고치거나 해서 이를 해결할수 있었습니다.

그래서 한숨한번 쉬고 구글링을 해서 sqlite db 를 뒤적 뒤적 하고 있었는데 해당 엔트리가 아예 없는 겁니다. 그래서 이거 없으면 만들어 넣어야 하나? 하고 고민하다 번뜩 드는 생각이 ..

혹시 4.1.x 대는 UI 로 나와 있는 것이 아닐까? 하는 생각이 들어 설정을 찾아보니 있군요!!!

제가 쓰는 것이 커펌이라 모든 젤리빈에 해당되는 지는 알수 없지만 적어도 제가 쓰는 a500의 커펌에서는 설정-wifi-메뉴버튼-고급(advanced) 를 들어가면 wifi-region code 설정이 가능하고 이걸 한국이나 일본으로 설정하면 해결됩니다. ( 그냥 미국 이외면 다 됩니다. )

다른 젤리빈은 어떤가 하고 젤리빈이 올라가 있는 제 옵티이이를 보니 이넘은 wifi 관련 항목이 sk 커스텀이군요. –;;;
