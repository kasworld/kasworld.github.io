한성 인민 에어 를 샀습니다.

By SeukWon Kang

December 7, 2013

수요일에 주문했는데 오늘(토)에야 도착하는 군요.

정식모델명은 한성컴퓨터 A34X ForceRecon 2467 (mSATA 250GB) 이고

간단한 스펙은 하스웰 i5 4250U ( 1.3Ghz ) 2c4t 에 hd 5000 그래픽 8G 램 ( 싱글 채널 ) 256G 삼성 830 ssd ( 이벤트로 대신 들어간 ) 1920x1080 13.3인치 스크린 입니다.

도착해서 포장을 풀고 무게를 재보니 1.399 kg 이더군요. 지금 다시 재보니 1.387 kg  입니다. ( 주방저울 정밀도가 그리 높지 않은듯 )

한성제품을 써보는건 이번이 네번째인데

전 직장에서 쓰던 X53-BossMonster Lv.77 지금 직장에서 쓰고 있는 X54-BossMonster LV.82 안드로이드 장난감으로 산 gboard 8.9 그리고 이번에 산 A34X ForceRecon 2467

다들 꽤 맘에 들어하며 사용했고/하고  있습니다.

물론 저는 저 네개의 머신에 모두 리눅스를 사용하고 있지요. ^^ ( 안드로이드도 리눅스 입니다. )

리눅스(xubuntu)에서 모든 기능이 지원 되는 것은 아니지만 ( x54와 a34 는 화면 밝기가 지원 되니 않네요. ) 필요한 모든 기능 ( 유/무선 랜, 사운드 , 터치 패드 , 블루투스 , 키보드 , 전원 관리 )이 잘 작동되니 큰 문제는 없습니다.

하스웰 버전의 인민 에어에 대한 간단한 평을 해보면

받자마자 ( os 없는 버전입니다. ) xubuntu를 설치하고 개인적으로 작업하는 설정들을 하고 나니.

조용하고 ( 소리가 거의 없습니다. ) 키보드 타자감 나쁘지 않고, ( 키 배치는 사기 전부터 쓸만해 보인다고 생각하고 있었습니다. 그 크기에서는 ) 의외로 터치패드 사용감이 좋은 편입니다. ( 다 애플의 공덕이지요. ) 다만 특이한게 터치패드가 (리눅스 드라이버를 보니)ps/2로 연결되며  오른쪽 클릭 버튼이 없습니다. ( 맥 처럼 패드 전체가 눌리는데 오른쪽을 눌러도 왼 클릭입니다. ) 오른클릭은 두손가락, 가운데 클릭은 세손가락 으로 클릭하면 됩니다.

나름 익숙해지니 쓸 만은 합니다.

크기는 거의 대놓고 맥북 에어를 벤치마킹한것이라 13인치 맥북에어랑  똑 같습니다. ( 바닥 라인이 달라서 케이스는 힘들겠지만 가방등의 액서세리는 간단히 호환될듯.. )

무선랜이 5G를 지원하지않아서 좀 걱정했는데 ( 아무래도 싸구려 랜 칩이 들어가니) 그래도 150M 연결로 잡히며 iperf로 테스트 해보니 95Mbit/sec 정도 속도를 뽑아줍니다.

혹시 무선랜이 느리면 6년 된 x61t의 랜카드와 바꿀까 생각했는데 ( 이건 5G 지원됩니다. ) 그럴 필요까지는 없을것 같네요.

성능은 7z b 테스트에서 7000 정도 나옵니다. ( 싱글채널 메모리라 성능 영향이 있을 껍니다. ) 참고로 i5 2500 ( 4c4t , 3.4Ghz ) 에선 13000 정도 입니다.

2013-12-09 추가분 밝기 조절은 기능 키로는 작동하지 않는데 xfce-power-manager-plugins 에 들어있는 brightness plugin을  사용하면 조절 됩니다. 웃긴건 보스 몬스터 82는 이걸로도 안되는 군요. 아니면 /sys/class/backlight/acpi_video0/brightness 에 echo 해 넣어도 됩니다. 전 슬립 들어갔다 나올때 마다 최대 밝기가 되는 게 짜증나서 ( 배터리도 많이 먹고) /etc/rc.local 파일에 echo 50 > /sys/class/backlight/acpi_video0/brightness 를 추가 했습니다. 보스 몬스터는 이것도 안먹는 군요.

그리고 마우스는 딸깍 할때 까지 누르면;; 오른쪽 클릭이 먹습니다. 또 회사에서 써보니 내장 랜이 공유기를 많이 타는 군요. 집에서는 아무 문제가 없었는데 회사 공유기는 잘 붙는 것과 아예 붙지 않는 것이 있습니다. 역시 x61t 랜카드를 옮겨 와야 할듯 합니다.
