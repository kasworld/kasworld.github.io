소니 엑스페리아 레이에 안드로이드 L (5.1.x)를 설치하기.

By SeukWon Kang

May 30, 2015

http://forum.xda-developers.com/xperia-mini/ray-development/rom-cyanidel-rc-115-1-1-t3105417

일종의 취미 생활인 가끔 생각나면 레이가지고 장난치기 입니다.

오랫만에 XDA를 들어가보니 레이용 안드로이드 L!! 이 나왔더군요.

그래서 설치하고 있습니다.

( rom flash 하는 동안 쓰는 글입니다. )

진행순서는

위 링크를 들어가서 cyanide-5.1-RC-11.0-urushi.zip HDPI_miniGAPPS-5.1.1-20150527-signed.zip System-Formating-Tool_signed.zip UPDATE-SuperSU-v2.46.zip

을 받아 놓습니다.

cyanide-5.1-RC-11.0-urushi.zip 안의 boot.img 를 풀어내고 .

fastboot모드로 진입합니다. ( 방법은 기계가 꺼진 상태에서 볼륨 위쪽 버튼을 누른 채로 pc 와 usb 케이블로 연결하면 됩니다. 홈 버튼 주변 이 파란색으로 빛나면 제대로 진입한 것입니다. )

boot.img를 flash 합니다. ( 기계가 보이는지 확인 fastboot devices 커널을 설치 ( 커널 파일 이름이 boot.img 입니다 ) fastboot flash boot boot.img 기계 재부팅 fastboot reboot)

그다음은 링크대로 CWM 모드로 진입한후 ( 부팅시 home 연타 ) ( 설치되어 있지 않으면 cwm 설치부터 해야 합니다. )

  1. WIPE DATA/FACTORY RESET
  2. Flash system formatting tool zip
  3. Wipe Cache
  4. Flash the ROM.
  5. Reboot wait for system settle down everything!!!!
  6. reboot again and flash the Gapps 7)flash Supersu zip

인데 5번에서 멈춰있군요.

뭐가 문제이려나요. ;;;

오래 걸리는 군요.

다시 링크 글을 잘 읽어보니 sdcard에 ext4 파티션이 있어야 하나 봅니다. sdcard를 큰거 (16Gbyte) 로 바꾸고 fat32 와 ext4 파티션을 가지도록 포맷 하고 ( 어느것을 앞으로 해야 하는지 몰라서 fat32, ext4 순으로 만들었습니다. ) flash할 zip 파일들을 fat32에 복사해 넣고 있습니다. ;;;

1번부터 다시 하고 있는데 5번에 나오는 메시지를 보니 잘 되는 것 같습니다. detected filesystem ext4 at 어쩌구 하고 나옵니다. ( 아까는 이부분에서 못찻는다는 에러가 나왔습니다. ) 아마도 레이의 내장 용량이 작으니 sdcard를 사용하게 하는 것 이겠지요.

아무튼 무사히 지나가서 6,7번도 진행했습니다. 부팅해보니 내장 저장공간이 (app install 공간 ) 넖어져서 좋군요.

이 모든 작업은 linux mint 17.1 에서 이루어 졌습니다. (windows 쓰시는 분들은 다른 방법을 찾아야 할듯 합니다. )

아 그리고 설치하고 나니 라디오가 없군요;;; 라디오 유용한데. ;;
