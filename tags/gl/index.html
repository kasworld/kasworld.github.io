그저 3d 공간에 공들이 돌아다닐 뿐인 프로젝트

https://github.com/kasworld/gowasm3dgame 꽤 예전에 서버 기반 게임 프레임웍 으로 만들었던 go4game을 remake 한 프로젝트 입니다. 서버 기반 게임 제작을 위한 프레임웍/라이브러리 들인 genprotocol 서버 클라이언트가 사용할 프로토콜 생성, 관리 argdefault : config와 command line arguments prettystring : struct 의 string 화 / admin web , debug용 genenum : enum 의 생성, 관리 log : 전용 log package의 생성, 사용 signalhandle : signal을 관리해서 프로그램의 linux 서비스화, start,stop,forcestart, 들을 사용해서 만들어 봤습니다.
