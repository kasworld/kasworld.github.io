리눅스 민트 14 시나몬을 설치했습니다.

며칠 전부터 고민하다가 오늘 일어나자마자 다 갈아업고 민트14를 설치했습니다. 잠깐 써본 느낌은 우분투 유니티에 짜증나고 xfce는 약간 부족하다고 느끼는 사람들에게는 아주 좋은 대안이 될것 같습니다. 사실 우분투가 이렇게 나왔어야 하지 않나 하는 생각이 들더군요. 처음 설치하면 nvidia 비디오 드라이버를 못잡아서 수동으로 설치해 줘야 하지만 금방 잘 잡히고 문제없이 작동합니다. 그리고 언제나처럼 제 손에 맞추기 위해 추가로 설치 및 설정한 것들을 기록합니다. 오라클의 뻘짓때문에 mysql대신 mariadb ppa 를 추가하고 설치 git / gvim 설치 및 설정 python 관련 모듈들 설치
