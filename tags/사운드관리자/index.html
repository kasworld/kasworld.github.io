2주 전에 구입한 40인치 모니터

2주 전에 구입한것은 라즈베리파이만은 아닙니다. 기존에 쓰던 3개의 모니터가 하나씩 하나씩 망가져서 15인치 모니터 하나 만을 사용하고 있었는데 노안 때문인지 눈이 너무 아파서(란 핑계로 마눌님을 설득하여 ^^ ) 큰 모니터를 구입하였습니다. 구매를 하기 위해 한달 정도 정보를 조사 해보았는데 예전에 쓰고 있던 델 27인치 2560x1440 모니터보다는 크고 픽셀이 많았으면 좋겠다고 생각하여 4k 모니터 로 ( 3840x2160 ) 로 대상을 선정하고 모니터를 놓을 공간 ( 책상위 )의 제약을 고려하니 ( 가로 폭이 책상폭 120cm 보다 작을것 )
